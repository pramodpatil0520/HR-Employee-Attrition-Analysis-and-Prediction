## 📌 Project Overview

This is a personal machine learning project built as part of my self-learning journey in **data analysis and ML**. The goal is to predict whether an employee at Sailfort Motors will leave the company (`left = 1`) or stay (`left = 0`) — a classic binary classification problem using real-world HR data.

The project covers the **complete ML pipeline**:
- Exploratory Data Analysis (EDA)
- Data Preprocessing & Feature Engineering
- Model Building & Hyperparameter Tuning
- Model Evaluation & Comparison
- Business Insights & HR Recommendations

## 📊 Dataset

| Property | Value |
|---|---|
| **Source** | HR Sailfort Motors Dataset |
| **Total Records** | 14,999 rows |
| **After Deduplication** | 11,991 rows |
| **Features** | 9 input features + 1 target |
| **Target** | `left` (0 = Stayed, 1 = Left) |
| **Class Distribution** | 83.4% Stayed / 16.6% Left (imbalanced) |
| **Missing Values** | None |

### Feature Description

| Column | Type | Description |
|---|---|---|
| `satisfaction_level` | Float | Employee self-reported satisfaction (0–1) |
| `last_evaluation` | Float | Last performance review score (0–1) |
| `number_of_projects` | Integer | Number of concurrent projects assigned |
| `average_monthly_hours` | Integer | Average working hours per month |
| `tenure` | Integer | Years spent at the company |
| `work_accident` | Binary | Had a work accident (0/1) |
| `left` | Binary | **TARGET** — Left company (1) or Stayed (0) |
| `promotion_last_5years` | Binary | Promoted in the last 5 years (0/1) |
| `department` | Categorical | Department (Sales, Technical, HR, IT, etc.) |
| `salary` | Categorical | Salary band (low / medium / high) |

---

## 🔍 Key EDA Findings

- **Satisfaction is the #1 driver** — employees with satisfaction < 0.15 overwhelmingly leave, even if they have high performance scores (frustrated high-performer pattern)
- **Burnout trap** — employees working 250–300+ hours/month on 6–7 projects form a distinct "leave" cluster
- **The 3–5 year quit window** — nearly 50% of year-5 employees left; year 3 has the highest departure volume
- **Low salary = high attrition** — majority of leavers are in the low salary band; high earners almost never quit
- **Promotion = retention shield** — less than 10 promoted employees left across the entire dataset
- **Sales & Technical** departments have the highest attrition counts

## 🤖 Models Built

### 1. Logistic Regression (Baseline)
- Solver: `saga`, Max iterations: 5000
- Tuning: `GridSearchCV` (5-fold CV)
- Parameters searched: `penalty` (l1, l2, elasticnet), `C` (1–50)

### 2. Random Forest Classifier
- Tuning: `RandomizedSearchCV` (4-fold CV, 20 iterations)
- Metric optimized: `ROC-AUC`
- Parameters searched: `n_estimators`, `max_depth`, `max_features`, `max_samples`, `min_samples_leaf`, `min_samples_split`

### 3. XGBoost Classifier ⭐ Best Model
- Objective: `binary:logistic`
- Class imbalance: `scale_pos_weight = n_negative / n_positive ≈ 4.0`
- Tuning: `RandomizedSearchCV` (4-fold CV, 20 iterations)
- Parameters searched: `n_estimators`, `max_depth`, `learning_rate`, `subsample`, `colsample_bytree`, `min_child_weight`, `gamma`

## 📈 Results

| Metric | Logistic Regression | Random Forest | XGBoost |
|---|---|---|---|
| Accuracy | ~83.0% | ~98.0% | ~98.5% |
| Precision (Left=1) | ~72.0% | ~96.0% | ~97.0% |
| Recall (Left=1) | ~61.0% | ~91.0% | ~93.0% |
| F1 Score | ~66.0% | ~93.5% | ~94.9% |
| **ROC-AUC** | ~0.810 | ~0.980 | **~0.986** |

> **XGBoost is the recommended model** — highest scores across all metrics, native class imbalance handling, and built-in regularization.

### Top 5 Most Important Features (XGBoost)
1. `satisfaction_level`
2. `average_monthly_hours`
3. `number_of_projects`
4. `last_evaluation`
5. `tenure`

## 💡 Business Recommendations

Based on the analysis, the top actions Sailfort Motors HR should take:

1. **Monthly satisfaction surveys** — flag employees scoring below 0.40 for 1:1 check-ins
2. **Workload monitoring** — alert when employees exceed 200 hrs/month or 5+ concurrent projects
3. **Year 3–5 retention program** — automatic career reviews and milestone rewards at the quit window
4. **Salary audit** — prioritize low-salary employees in Sales and Technical departments
5. **Transparent promotion pipeline** — communicate clear criteria; promotions dramatically reduce attrition
